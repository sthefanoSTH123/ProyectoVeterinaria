1. Objetivo

Desarrollar un módulo que permita registrar y administrar las consultas realizadas en la veterinaria, relacionando cada consulta con un cliente y una mascota determinados.

2. Datos de la Consulta

El sistema deberá almacenar la siguiente información:

Campo	Tipo
IdConsulta	Int
Fecha	Fecha
Cliente	Cliente
Mascota	Mascota
Motivo	Texto
Diagnostico	Texto
Costo	Double

3. Funcionalidades
El sistema deberá permitir registrar una nueva consulta.
El sistema deberá mostrar todas las consultas registradas en una tabla.
El sistema deberá permitir editar la información de una consulta seleccionada.
El sistema deberá permitir eliminar una consulta seleccionada.
El sistema deberá permitir seleccionar un cliente registrado.
El sistema deberá permitir seleccionar una mascota registrada.
El sistema deberá almacenar el motivo de la consulta.
El sistema deberá almacenar el diagnóstico emitido durante la atención.
El sistema deberá registrar el costo de la consulta.
