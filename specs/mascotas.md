1. Objetivo

Desarrollar un módulo que permita administrar la información de las mascotas registradas en la veterinaria, facilitando el registro, consulta, modificación y eliminación de datos, así como la asociación de cada mascota con su respectivo dueño.

2. Datos de la Mascota

El sistema deberá almacenar la siguiente información:

Campo	Tipo
Código	Int
Nombre	Texto
Edad	Int
Especie	Texto
Raza	Texto
Sexo	Texto
Peso	Double
FechaNacimiento	Fecha
Dueño	Cliente

3. Funcionalidades
El sistema deberá permitir registrar una nueva mascota.
El sistema deberá mostrar todas las mascotas registradas en una tabla.
El sistema deberá permitir editar la información de una mascota seleccionada.
El sistema deberá permitir eliminar una mascota seleccionada.
El sistema deberá permitir asociar una mascota a un cliente registrado.
El sistema deberá permitir seleccionar la especie de la mascota desde una lista predefinida.
