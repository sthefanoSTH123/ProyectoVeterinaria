1. Objetivo
Desarrollar un módulo que permita administrar la información de los clientes de la veterinaria, facilitando el registro, consulta, modificación y eliminación de datos.
2. Datos del Cliente

El sistema deberá almacenar la siguiente información:

Campo	Tipo
Cédula	Int
Nombre	Texto
Apellido	Texto
Telefono	Int
Correo	Texto
Direccion	Texto

3. Funcionalidades
El sistema deberá permitir registrar un nuevo cliente.
El sistema deberá mostrar todos los clientes registrados en una tabla.
El sistema deberá permitir editar la información de un cliente seleccionado.
El sistema deberá permitir eliminar un cliente seleccionado.
